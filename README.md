# Trail Framework

Trail is an artifact-driven framework for executing work — with humans, AI, or both — without letting the conversation become the source of truth.

**Documentation:** [trail.venturanomadica.com](https://trail.venturanomadica.com)

---

## What's in this repo

- `docs/` — Trail Framework documentation (rendered at trail.venturanomadica.com/docs)
- `trail/` — The Trail scaffold — copy this into your project to get started

## Getting started

Download the scaffold and unzip it into your project:

[Download trail.zip](https://github.com/Ventura-Nomadica/trail-framework/releases/latest/download/trail.zip)

Or clone and copy the `trail/` folder directly.

Full quickstart: [trail.venturanomadica.com/docs/quickstart](https://trail.venturanomadica.com/docs/quickstart)
